    Simple task manager

    Simple task manager, made with using of localStorage. You can add you name, create tasks, mark them as done and delete all tasks.

    To add a new task, put cursor inside input, write you task and press Enter.

    Click on task switches them between done/not done (green and orange colors)

    X - button will delete all tasks. So be carefull with it!
